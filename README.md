# Password_Authentication

A simple and secure Password Authentication System that allows users to register and login using encrypted passwords. This project demonstrates the basic implementation of authentication including password hashing and validation.

Features

User Registration

User Login

Password Encryption (Hashing)

Input Validation

Simple and beginner-friendly

Build Tool: Gradle

How It Works

User enters a password during registration.

The password is converted into a hashed form.

The hashed password is stored securely.

During login, the entered password is hashed again.

If both hashes match, authentication is successful.
